# Social Media Performance Analysis - Frontend

## Overview

This repository contains the frontend for the **Social Media Performance Analysis** project. The frontend is built using **React** and **Tailwind CSS**. It provides a dashboard where users can view and analyze the performance of their social media posts. It interacts with the backend to fetch engagement data, process it, and display detailed insights.

## Features

- Displays key performance metrics like likes, shares, and comments for social media posts.
- Provides a visual representation of engagement data (graphs, charts, etc.).
- Allows users to view personalized insights based on post performance.
- Provides authentication for users to log in and manage their profile.
- Built with **React** for dynamic components and **Tailwind CSS** for a responsive design.

## Technologies Used

- **React**: JavaScript library for building user interfaces.
- **Tailwind CSS**: Utility-first CSS framework for fast styling.
- **Axios**: HTTP client for making requests to the backend API.
- **Chart.js**: JavaScript library for displaying engagement data in charts.
- **React Router**: Library for handling page navigation.
